Politique de confidentialité et de cookies

Dernière mise à jour : 10 mai 2026

1. Introduction

Le présent site web a pour objectif de présenter les projets, réalisations et informations professionnelles de son propriétaire.

Le respect de votre vie privée est une priorité. Cette politique explique quelles données peuvent être collectées lors de votre navigation, comment elles sont utilisées, ainsi que vos droits concernant ces informations.

2. Responsable du traitement des données

Responsable du site : Antoine Smeets

Site web : https://smeets.dev

Pour toute question relative à la confidentialité ou au traitement des données, vous pouvez communiquer via les moyens de contact disponibles sur ce site.

3. Données collectées

a) Données de navigation anonymisées

Ce site utilise Plausible Analytics, une solution d’analyse web respectueuse de la vie privée, auto-hébergée par le propriétaire du site.

Les statistiques collectées servent uniquement à :

  • comprendre l’utilisation générale du site ;
  • améliorer le contenu et les performances ;
  • analyser les pages et projets les plus consultés.

Les données analysées peuvent inclure :

  • les pages visitées ;
  • la provenance approximative du trafic ;
  • le type d’appareil ou de navigateur ;
  • la durée approximative des visites ;
  • les interactions générales avec le site.

Aucune donnée n’est utilisée à des fins publicitaires ou de profilage marketing.

4. Cookies et consentement

Ce site utilise un mécanisme de consentement pour les statistiques d’audience.

Lorsque vous acceptez les statistiques :

  • le script Plausible Analytics est chargé ;
  • un choix de consentement peut être conservé localement dans votre navigateur afin d’éviter de vous redemander votre préférence à chaque visite.

Lorsque vous refusez :

  • aucune statistique d’audience n’est collectée ;
  • aucun script analytique n’est chargé.

Le stockage de cette préférence peut utiliser :

  • le localStorage du navigateur ;
  • ou un cookie technique minimal nécessaire au fonctionnement du consentement.

5. Absence de publicité et de suivi marketing

Ce site :

  • n’utilise pas Google Analytics ;
  • n’utilise pas Meta Pixel, TikTok Pixel ou autres outils publicitaires similaires ;
  • ne pratique pas le reciblage publicitaire ;
  • ne revend pas les données des visiteurs.

6. Hébergement et sécurité

Les services liés au site et aux statistiques peuvent être hébergés sur une infrastructure contrôlée par le propriétaire du site ou par des fournisseurs d’hébergement tiers.

Des mesures raisonnables sont mises en place afin de protéger les données contre :

  • l’accès non autorisé ;
  • la divulgation ;
  • la perte ou la modification.

7. Conservation des données

Les données analytiques sont conservées uniquement pour une durée raisonnablement nécessaire à l’analyse du trafic et à l’amélioration du site.

8. Vos droits

Conformément aux lois applicables, notamment la Loi 25 au Québec, vous pouvez :

  • demander des informations concernant les données vous concernant ;
  • demander la rectification ou la suppression de certaines données ;
  • retirer votre consentement aux statistiques à tout moment.

Vous pouvez également configurer votre navigateur afin de limiter certains mécanismes de stockage local ou de cookies.

9. Modifications de cette politique

Cette politique peut être modifiée à tout moment afin de refléter l’évolution du site, des technologies utilisées ou des exigences légales.

La version la plus récente sera toujours accessible sur cette page.
